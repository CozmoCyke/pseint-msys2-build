\b;Instruction \c;open\n;
L'instruction \c;open();\n; ouvre un fichier de texte dans le dossier 'files/'. Cette instruction est une méthode de la classe \c;\l;file\u cbot\file;\n;. C'est la raison pour laquelle on n'écrit jamais \c;open()\n; tout seul, mais \c;handle.open()\n;:
\c;
\s;	handle.open("essai.txt", "w");
\n;
Pour ouvrir un fichier, procédez comme suit:
\c;
\s;	file handle();
\s;	handle.open("filename", "w");
\s;	handle.writeln("abc");
\s;	handle.close();
\n;
\c;"r"\n; mode lecture : fichier ouvert pour lecture. 
\c;"w"\n; mode écriture: fichier ouvert à l'écriture.\c;"a"\n; mode mise à jour: fichier ouvert pour ajout/modification.

Les fichiers ne peuvent être créés et ouverts que dans le dossier 'files/', situé là où Colobot a été installé. Il n'est pas possible de créer ou ouvrir des fichiers situés ailleurs que dans ce dossier 'files/'.
