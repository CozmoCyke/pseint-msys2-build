# PPP

## Localization architecture

The project uses a functional separation for localization resources:

- `lang/` = program UI strings
- `keywords/` = localized syntax and runtime keyword tables
- `bin/help/` = long-form runtime help
- `bin/ejemplos/` / `ejercicios/` = localized pedagogical content

See `docs/syntax-localization-architecture.md` for the official architecture and evolution rules.
