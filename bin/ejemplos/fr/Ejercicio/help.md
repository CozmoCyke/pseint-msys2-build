### Structures de controle

On veut obtenir un algorithme pour calculer le montant total a payer pour un achat de plusieurs articles :

* Completez le pseudo-code afin que, apres avoir demande le nombre d'articles, il permette de saisir le prix de chacun puis affiche le montant a payer.
* Le montant a payer, qui doit etre stocke dans la variable `monto`, est egal a la somme du prix de tous les articles si cette somme est inferieure a $500. Si elle est egale ou superieure a $500, une reduction de $80 est appliquee.
* Par exemple, pour un achat de 5 articles dont les prix sont $90, $70, $150, $180 et $200, l'utilisateur doit d'abord saisir 5, puis 90, 70, 150, 180 et 200. Le total est de 90+70+150+180+200=690. Comme le total depasse $500, la reduction est appliquee, donc le montant a payer est de $690-$80=$610.
* **N'ajoutez aucun message supplementaire** (des choses comme *"entrez le prix unitaire"*), et ne modifiez ni ne supprimez les lignes de pseudo-code existantes. Vous devez seulement completer l'algorithme pour effectuer les lectures manquantes et calculer la valeur de `monto`.
