### Control Structures

Create an algorithm that calculates the total amount to pay for a purchase of several items:

* Complete the pseudocode so that, after asking for the number of items, it lets the user enter the price of each one and finally shows the amount to pay.
* The amount to pay, which must be stored in the variable `monto`, is the sum of all item prices when that sum is below $500. If it is $500 or more, a discount of $80 is applied.
* For example, for a purchase of 5 items priced at $90, $70, $150, $180 and $200, the user should first enter 5 and then 90, 70, 150, 180 and 200. The total is 90+70+150+180+200=690. Since the total is above $500, the discount is applied, so the amount to pay is $690-$80=$610.
* **Do not add any extra prompt messages** (things like *"enter the unit price"*), and do not modify or remove the existing pseudocode lines. You only need to complete the algorithm so it performs the missing reads and computes the value of `monto`.
