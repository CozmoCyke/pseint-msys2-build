Arrays

   Arrays are homogeneous data structures (all their values have the same type) that let you store a certain number of values under a single identifier and later refer to them using one or more subscripts. Arrays can be thought of as vectors, matrices, and similar structures. To use an array, it must first be dimensioned; that is, you must declare the ranges of its subscripts, which determines how many elements it stores and how they are accessed.

Arrays

   The Dimensionar instruction (or Dimensión) declares an array by indicating its dimensions.

      Dimensionar <identificador>[<max1>,...,<maxN>];

   This instruction defines an array with the name given in <identificador> and N dimensions. The N parameters indicate the number of dimensions and the maximum value of each one. The number of dimensions may be one or more, and each maximum number of elements must be a positive numeric expression.
   If zero-based arrays are used, declaring an array with, for example, 15 elements means the valid indices go from 0 to 14; if one-based arrays are used, the valid indices go from 1 to 15. This is configured in the PSeudoCode Options dialog.
   More than one array may be declared in the same instruction, separating them with a comma (,).

      Dimensionar Vec[10], Mat[5,8]

   The Largest Values example shows how to load data into an array and then find the greatest value among those data.
   The Branches example uses a two-dimensional array (matrix) to store a company's sales broken down by product and branch office.

Arrays

   Depending on the language customization, it may be possible to resize an array with the keyword Redimensionar (or Redimensión). When an array is resized, the original number of dimensions must be preserved, but the size of each dimension may change.

      Dimensionar Lista[10];
      // ...
      Redimensionar Lista[20];
      // ...
