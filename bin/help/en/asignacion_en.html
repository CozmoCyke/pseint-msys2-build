Assignment


   The assignment instruction stores a value in a variable.

      <variable > <- <expresión> ;

   When an assignment is executed, the expression on the right is evaluated first and then the result is assigned to the variable on the left. The type of the variable and the type of the expression must match.

   If the variable on the left did not exist previously, it is created. If it already existed, its previous value is lost and replaced with the new value, which is why assignment is said to be destructive. The contents of the variables used in the expression on the right are not modified.

   There are two alternative assignment operators that may be used in any case, although the second one (=) depends on the selected language profile.

      <variable> := <expresión> ;
      <variable> = <expresión> ;
