Help.html Format

In this file you should use a small subset of HTML. In particular, some useful tags are: H1/H2/H3/H4 (titles of different levels), B (bold), UL (unordered list), OL (ordered list), LI (list item), I (italic), BR (line break), IMG (image), TABLE (table), TR (table row), TD (table cell), CODE (special font for code), U (underline), CENTER (centered), and HR (horizontal line).

Also, some special characters are interpreted by the HTML parser instead of being shown as text. In particular, to insert the sign < you must use "&lt;", for the sign > use "&gt;", for & use "&amp;", and for extra spaces use "&nbsp;".

To insert an image, use the IMG tag and place the image path in its src attribute. In a self-contained exercise you must add the prefix "memory:" to the image path (file name), and use only png images.
