Data Types

  • Simple Types: Numeric, Logical, Character.
  • Data Structures: Arrays.
  • Explicit variable definition.


    Simple data types are determined automatically when variables are created. The two actions that can create a variable are input (READ) and assignment (<-). For example, the assignment "A<-0;" implicitly indicates that variable A will be numeric. Once the data type has been determined, it must remain constant during the whole execution of the process; otherwise the process will be interrupted.

    The type of a variable can be defined before it is used. This definition can be mandatory or optional depending on the language configuration.

    Arrays are homogeneous structures (that is, all of their elements are of the same simple data type). The type of their elements is determined when one of them is used, in the same way as for simple-type variables. They can be one-dimensional or multidimensional, but they must be dimensioned before being used for the first time through the DIMENSION directive (example: "DIMENSION A[10,20];").

    Identifiers, or variable names, must contain only letters, numbers, and/or underscore (_), always beginning with a letter.
