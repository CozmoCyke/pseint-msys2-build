Simple Types

   There are three basic data types:
  • Numeric: numbers, both integers and real numbers. A dot is used as the decimal separator. Examples: 12 23 0 -2.3 3.14
  • Logical: it can only take two values: VERDADERO or FALSO.
  • Character: characters or strings enclosed in quotes (single or double). Examples: 'hola' "hola mundo" '123' 'FALSO' 'etc'


    Simple data types are determined automatically when variables are created. The two actions that can create a variable are input (LEER) and assignment (<-). For example, the assignment "A<-0;" implicitly indicates that variable A will be numeric. Once the data type has been determined, it must remain constant during the whole execution of the process; otherwise the process will be interrupted.

    Optionally, a numeric variable can be declared as integer with the DEFINIR instruction. In that case, any non-integer value read or assigned to it is truncated.
