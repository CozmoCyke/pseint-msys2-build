Input


   The Read instruction lets you enter information from the environment.

      Leer <variable1> , <variable2> , ... ,
<variableN> ;

   This instruction takes N values from the environment (in this case, the keyboard) and assigns them to the N variables listed. One or more variables may be included, so the command reads one or more values.

   If a variable where the read value must be stored does not exist, it is created during the read. If the variable already exists, its previous value is lost because it takes the new value, which is why input is said to be destructive.

   If flexible syntax is used, the variables to be read may optionally be separated with spaces instead of commas. This is configured in the PSeudoCode Options dialog.

   The Sum example shows a very simple program that reads two numbers and calculates and displays their sum.
