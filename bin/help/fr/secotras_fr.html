Others

   In addition to input, output, and assignment, PSeInt includes some other simple sequential instructions such as waiting, clearing the screen, or dimensioning arrays.

  • Esperar: pauses execution for a given amount of time.
  • Borrar Pantalla: clears the execution screen.
  • Dimensionar: declares arrays and indicates their dimensions.
  • Redimensionar: resizes an array if the current profile allows it.

The instructions for arrays are described in more detail in the corresponding section.
