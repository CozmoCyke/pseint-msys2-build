Information for Teachers

This section explains how to perform certain actions in PSeInt that are generally needed only by teachers, not by students:

  • Profile Creation: Profiles allow certain language rules to be adjusted to each teacher's needs. PSeInt includes a broad list of preconfigured profiles for different educational institutions. If an institution is not included in the list, this link explains how the teacher can create a custom profile file, distribute it to students, and request its inclusion in future versions of PSeInt.
    • Language customization (what profiles are and which characteristics can be customized)
    • Creating new profiles (steps to create and distribute a new profile, Information for Teachers)

  • Self-contained Exercise Creation: The exercise mode lets the teacher generate a single file with all the elements needed to present a complete exercise to the student, and the student's proposed solution is evaluated automatically with a predefined set of test cases.
    • Self-contained exercises (description of the feature from the student's point of view)
    • Creating new exercises (steps to create an exercise, Information for Teachers)
