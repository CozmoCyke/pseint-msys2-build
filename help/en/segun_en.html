   The sequence of instructions executed by an Segun instruction depends on the value of a numeric variable.

      Segun <variable> Hacer
            <número1>: <instrucciones>
            <número2>,<número3>: <instrucciones>
            <...>
            De Otro Modo: <instrucciones>
      FinSegun


   This instruction lets you execute one of several possible actions, depending on the value stored in a numeric variable. When it runs, the variable is evaluated and the sequence of instructions associated with that value is executed.
   Each option is formed by one or more numbers separated by commas, a colon, and a sequence of instructions. If an option includes several numbers, its instruction sequence is executed when the value of the variable matches any of those numbers.
   Optionally, you can add a final option called De Otro Modo, whose associated instruction sequence is executed only if the value stored in the variable does not match any of the previous options.

   The Menu example shows a very simple program that uses this control structure to choose what to display on screen according to the option selected by the user from a menu.
