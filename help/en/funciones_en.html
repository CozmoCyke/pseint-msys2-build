Predefined Functions

   Functions in pseudocode are used in a way similar to other languages. Their name is written followed by the arguments in parentheses (for example trunc(x)). They may be used inside any expression, and when the expression is evaluated they are replaced by the corresponding result. At present, all the available functions are mathematical or utility functions. The built-in functions available are listed below:

          
Function Meaning
RC(X) o Raiz(X) Square Root of X
Abs(X) Absolute Value of X
Ln(X) Natural Logarithm of X
Exp(X) Exponential Function of X
Sen(X) Sine of X
Cos(X) Cosine of X
Tan(X) Tangent of X
ASen(X) Arcsine of X
ACos(X) Arccosine of X
ATan(X) Arctangent of X
Trunc(X) Integer Part of X
Redon(X) Closest Integer to X
Azar(X) Random integer in the range [0;x-1]
Aleatorio(A,B) Random integer in the range [A;B]
Longitud(S) Number of characters in string S
Mayusculas(S) Returns a copy of S with all characters in uppercase
Minusculas(S) Returns a copy of S with all characters in lowercase
Subcadena(S,X,Y) Returns a new string consisting of the part of S that goes from position X to position Y, including both ends. Positions use the same base as arrays, so the first letter is 0 or 1 depending on the selected language profile.
Concatenar(S1,S2) Returns a new string obtained by joining S1 and S2.
ConvertirANumero(X) Receives a string containing a number and returns a numeric value with that number.
ConvertirATexto(S) Receives a real value and returns its text representation as a string.
HoraActual() Returns a 5- or 6-digit integer in HHMMSS format, where HH is the current system hour, MM the minutes, and SS the seconds.
FechaActual() Returns an 8-digit integer in YYYYMMDD format, where YYYY is the year from the system date, MM the month, and DD the day.


Notes:
  • Modulo (the remainder of integer division) is not listed among the functions because it was implemented as an operator.
  • Trigonometric functions receive the angle in radians. To simplify conversions you can use the constant PI (for example, if A is an angle in degrees, its cosine is obtained with "cos(A*PI/180)").
  • Functions that operate on strings are available only if the selected language profile allows them (see PSeudoCode Options).
  • To create your own functions in pseudocode and use them from the main procedure, see the SubProcesses section.
  • To split the numbers returned by FechaActual and HoraActual, you can use the Mod operator and/or the Trunc function as shown in the FechaHora example.
