Quick Help for the Flowchart Editor

  • Move the view:
    • drag with the left button from any empty background area
  • Change the zoom:
    • rotate the mouse wheel
  • Add a new entity:
    • move the mouse to the left margin, select it, and drag it with the left button
    • note: some menu options change while Shift is pressed
  • Move an entity:
    • drag it with the left button
  • Edit the text of an entity:
    • double-click with the left button, or single-click with the right button
  • Delete an entity:
    • drag it with the left button to the lower-left corner
  • Duplicate an entity:
    • drag it with the right button while holding Shift
  • Select multiple entities:
    • drag with the left button from any empty background area
  • Create/delete/select subprocesses/subalgorithms:
    • first button on the toolbar (top left)
    • note: it is shown only if the profile allows it
