Help


Welcome to the PSeInt help system. This help is organized into the following sections:

  • Interface: Describes the main interface elements (buttons, menus, dialog boxes, etc.).
  • PSeudoCode: Describes the main characteristics of the interpreted pseudocode, its primitives, structures, and so on.
  • Algorithm Examples: Contains a list of the example algorithms included with the program and their descriptions.
  • Algorithm Tracing: Briefly introduces step-by-step execution for algorithm tracing and automatic desk checks.
  • Information for Teachers: Describes actions that normally only teachers, not students, need to perform.
  • Notes about PSeInt: License, change log, contact information, and more.


You can quickly search the help using the text box located at the top of this window.

Tip: In the main window, you can access the help for a keyword from the pseudocode editor by placing the text cursor on it and pressing Shift+F1.

If you cannot find what you are looking for, you can use this link to access the online forums and ask your question.
