Profile Creation

PSeInt allows teachers to customize some aspects of the language. To avoid confusion for students, the first time the program is run it asks the student to choose a configuration profile for their institution or course, quickly defining the language settings in a single step according to the teacher's requirements.

If your institution's profile is not among the ones already included in PSeInt, you can create it with the following steps:
  1. Select the item "Language Options (profiles)..." in the "Configure" menu.
  2. In the dialog box that appears, click "Customize".
  3. In the new dialog box, configure the profile as you wish. For more details about each available option, see Language Customization.
  4. Click the second button in the lower-left corner of the dialog box (the one showing a floppy disk) to save the profile to a file.
  5. Choose any location for the file and enter a brief description of the profile. It is recommended to mention the institution where it is used, the course or degree program, and the name of the responsible teacher or of the person sending the profile. This is the description shown at the bottom of the profile-selection window, and it is also used by the search box.
  6. There are two ways to distribute the generated profile so other users can use it:
    • The recommended way is to send the file by email to zaskar_84<arroba>yahoo.com.ar so it can be included in the list of built-in profiles for the next version of PSeInt. It is also recommended to attach a 24x24 pixel icon to display in the profile list and make it easier to find there. If you do not include one, the favicon from the institution's website will be used when possible.
    • You may also send the generated file directly to other users so they can load it from the profile customization window. This action is needed only once, because the customization remains saved in PSeInt's settings.


If you are a teacher and believe it would be useful to add more customizations beyond the ones already implemented, feel free to suggest them in your email when sending your profile. It is also recommended to include, in addition to the generated file and icon, information about the course where it is used (name of the university and faculty or institute, address, degree program, course name, teacher in charge, which real programming language is used in class after the pseudocode stage, etc.). This additional information is used for statistical purposes and possibly to complete the profile description if it is not sufficient. In no case will the teacher's email address or other personal information be published, except for the teacher's full name, which may be mentioned as the institution's teacher and as the profile's author.
