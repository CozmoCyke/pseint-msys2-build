Flowchart Editing

Quick help...


    PSeInt allows you to edit an algorithm by modifying either its pseudocode or its flowchart. To use the second method, open the pseudocode of the algorithm you want to edit and click the item "Edit flowchart..." in the "File" menu, or the corresponding toolbar button. This opens a new window where you can view and edit the diagram.

The toolbar in the flowchart editor window allows you to:
  • (F3): show the list of subprocesses of the current algorithm, choose which one to display and edit in the diagram, or add/delete subprocesses.
  • (F9): regenerate the pseudocode from the diagram to reflect the changes and try to execute it. If the generated algorithm is correct, it is executed; otherwise errors are marked with red icons on the entities that produce them and short descriptions are shown when you place the mouse over them.
  • (F5): apply the changes made in the flowchart editor to the pseudocode in the main PSeInt window and start step-by-step execution.
  • : export the diagram as an image file. A dialog box lets you modify the options used to generate the image.
  • (F12): fit the flowchart so it is fully visible in the window.
  • (F11): switch to fullscreen mode or return to windowed mode.
  • : switch between the available diagram styles. To choose the default type, use the corresponding option in the Configure menu of the main PSeInt window.
  • : show or hide the comments in the algorithm.
  • : enable or disable cropped text mode, which shows only the first letters of each entity to keep long instructions from distorting the diagram too much.
  • : enable or disable coloring entities according to their type.
  • (F1): show this help window.
  • (Esc): close the editor, asking whether the pseudocode in the main window should be updated if there are unapplied changes.

    To add entities (instructions or control structures) to the diagram, select them from the entity bar that appears when you move the mouse to the right margin of the window, and drag them to the position where you want to insert them. Some entities have variants. For example, a comment may be attached to a point in the flow or to another instruction; Repetir-Mientras Que is the alternative to Repetir-Hasta Que; and Para Cada is a variant of the conventional Para. To insert a variant, keep the Shift key pressed while selecting the entity.

    You can also duplicate an entity by dragging it while holding Shift.

    To edit the text of an entity, double-click it with the left mouse button or single-click it with the right button.

    To delete an entity, drag it to the trash bin in the lower-left corner of the window. This trash bin is shown only while an entity is being dragged.

    To create a multiple selection, hold Shift and drag with the left mouse button. This action defines a rectangular area and groups any logically contiguous set of entities contained within it. A dashed border appears around them. That means that if you try to move, duplicate, or delete one of them, the action is actually applied to the whole group.

    You can use the mouse wheel to change the zoom gradually, or select an area with the middle button to zoom into that area. To move the view, drag the background with the left or right mouse button. You can also switch to fullscreen mode with F11.

You can also use F7 to switch between view-only and edit modes. In view-only mode you cannot modify the algorithm. While editing, you can pin the entity panel on the right so it remains visible by using the Tab key.

    At any moment, if the algorithm has no errors, you can see its pseudocode conversion by returning the focus to the main PSeInt pseudocode editor. When you click the pseudocode, it is updated to reflect the changes generated in the diagram.

    Finally, note that errors are marked on the diagram only when the pseudocode is regenerated in the main window, not automatically after every change as happens in the code editor when the "Check syntax while typing" option is selected. However, you can regenerate the pseudocode at any time without running the algorithm by using F2.
