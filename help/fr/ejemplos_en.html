Algorithm Examples

    PSeInt includes a set of algorithms with different difficulty levels to illustrate the syntax and use of pseudocode. The available examples are described below:
  • Sum: The simplest example. It shows how to read two numbers into two variables, calculate their sum, and display it.
  • Average: Basic example of using an accumulator and the FOR structure to calculate the average of a set of values.
  • Guess the Number: Simple game in which the user must guess a random number.
  • Triangle: Determines from the lengths of three sides whether they form a right triangle and, if so, reports its area.
  • Largest Values: Finds the two largest values in a list of N data.
  • Sort List: Stores a list of names in an array and then sorts them alphabetically.
  • Modulo: Trivial example of using the modulo operator (MOD or %).
  • Menu: Simple menu example using the Repeat-Until and According To structures.
  • Digits: Separates the digits of an integer using the modulo operator.
  • Quadratic Formula: Uses the quadratic formula to compute the roots of a second-degree equation from its three coefficients.
  • Primes: Iteratively calculates the first N prime numbers.
  • Factorization: Example algorithm to decompose a number into its prime factors.
  • Branches: Uses arrays to store prices and quantities sold at different company branches, and then computes some statistics.
  • Strings: Shows the use of string functions to count the vowels in a sentence.
  • For: Traverses an array using the three variants of the For structure.
  • Colloquial: Shows some of the variants available in the Flexible language profile to make pseudocode closer to colloquial language.
  • SubProcesses/Functions: Basic example illustrating the syntax for declaring and invoking new functions in pseudocode.
  • Recursion: Example of a recursive function to calculate a power.
  • SubProcesses and Arrays: Example of creating subprocesses that receive arrays as arguments.
  • Missile: Animation example using ASCII codes and the Wait instruction.
  • Roman Numerals: Shows how to convert integers to Roman numerals using arrays.
  • Mathematics: Shows the use of expressions, operators, and mathematical functions through a menu-driven example.
  • Hanoi: Implements the well-known Towers of Hanoi game in pseudocode.
  • Tic-Tac-Toe: Algorithm for the Tic-Tac-Toe game.
  • Date and Time: Example of obtaining the current system date and time and splitting each into its parts.


Note: the syntax of these examples is adapted to the currently selected language configuration.
