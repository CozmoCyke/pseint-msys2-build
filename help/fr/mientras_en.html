While


   The Mientras instruction executes a sequence of instructions while a condition remains true.

      Mientras <condici¾n> Hacer
            <instrucciones>
      FinMientras

When this instruction runs, the condition is evaluated. If it is true, the sequence of instructions that forms the loop body is executed once. At the end of the loop body, the condition is evaluated again and, if it is still true, execution repeats. These steps continue while the condition remains true.
   Note that the instructions in the loop body may never execute if the condition is false the first time it is evaluated.
   If the condition is always true, this instruction creates an infinite loop. To avoid that, the loop body must contain some instruction that modifies one or more variables involved in the condition, so that it eventually becomes false and the loop ends.

   The Guess the Number example gives the user 10 attempts to guess a randomly generated number, using this structure to check whether the user guessed correctly or whether the attempts run out.
